# Fontendo

![Fontendo Logo](https://github.com/retrohead/Fontendo/blob/main/_src/Fontendo/Resources/Fontendo.png?raw=true)

Port of NintyFont by hadashisora to c#. All credit goes to him for the original application.


https://github.com/hadashisora/NintyFont

## Features

Currently supports loading and modifying CTR fonts only for now.

## Supported Formats

- CFNT - Binary Citrus FoNT, used in many 3DS titles utilizing old versions of NW4C.

## Download

Latest stable version usually compiled into the repo here: https://github.com/retrohead/Fontendo/tree/main/Fontendo/
